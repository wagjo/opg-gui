# OPG GUI


Tento repozitár je súčasťou výukových materiálov k predmetu [Objektovo Orientované Programovanie](https://oop.wagjo.com/). Projekt je určený pre použitie vo vývojovom prostredí IntelliJ IDEA 

Repozitár obsahuje príklady GUI aplikácie v AWT, Swing a Java FX.
